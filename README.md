#  AI Recruitment System

##  Overview
This project is an AI-assisted recruitment system designed to streamline the hiring process by intelligently evaluating and ranking candidates.

It uses Natural Language Processing (NLP) techniques such as TF-IDF vectorization and cosine similarity to compare candidate profiles with job descriptions. The system also combines multiple recruitment factors into a weighted scoring model to generate a ranked shortlist of candidates.

---

##  Features
-  Synthetic recruitment dataset generation  
-  Job description analysis  
-  TF-IDF based text vectorization  
-  Cosine similarity for candidate-job matching  
-  Weighted scoring system for evaluation  
-  Automated candidate ranking and shortlisting  

---

##  Tools & Technologies Used
- Python  
- Google Colab  
- Pandas  
- NumPy  
- Scikit-learn (TF-IDF, Cosine Similarity)  
- Matplotlib / Seaborn (for visualization)  

---

##  How It Works
1. A job description is defined.  
2. Candidate data is created or loaded.  
3. Text data is converted into numerical form using TF-IDF.  
4. Cosine similarity is calculated between candidates and job description.  
5. Additional factors are applied using a weighted scoring model.  
6. Candidates are ranked based on final scores.
